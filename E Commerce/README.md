# Small Ecommerce Website

This project is a small ecommerce website developed from scratch using HTML5, CSS3, and JavaScript. The website provides a user-friendly interface for browsing and purchasing products.

## Features

- Product Listing: Displays a collection of products with details such as name, price, and description.
- Product Details: Shows detailed information about a selected product, including images and specifications.
- Cart Functionality: Allows users to add products to a shopping cart and proceed to checkout.
- Fetch API Integration: Utilizes the Fetch API to retrieve product data from a fake API and dynamically update the website content.

## Technologies Used

- HTML5: Markup language for structuring the website.
- CSS3: Stylesheets for visual presentation and layout.
- JavaScript: Provides interactivity and dynamic behavior.
- Fetch API: Used to fetch data from a fake API and update the website content.

## Getting Started

To run the small ecommerce website locally, follow these steps:

1. Clone the repository to your local machine.
2. Open the index.html file in a web browser.
3. Explore the website and interact with the available features.
